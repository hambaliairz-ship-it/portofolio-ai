
## 1. App Flow (Alur Pengguna)

1. **Entry Point:** Rekruter membuka URL landing page Anda.
2. **Engagement:** Rekruter melihat *Hero Section* (siapa Anda) dan menelusuri proyek-proyek Anda.
3. **Validation:** Rekruter melihat daftar *skills* dan pengalaman kerja untuk memvalidasi kualifikasi.
4. **Conversion (The WhatsApp Part):** * Pengguna mengklik tombol "Chat via WhatsApp" atau mengisi form kontak sederhana.
* Sistem secara otomatis membuka aplikasi WhatsApp dengan pesan *pre-filled* (pesan otomatis) seperti: *"Halo [Nama Anda], saya melihat portofolio Anda dan tertarik untuk mendiskusikan peluang kerja..."*

---

## 2. Struktur Section Landing Page

Agar terlihat profesional, gunakan urutan section berikut:

| Section | Konten Utama |
| --- | --- |
| **Navbar** | Logo/Nama, Link navigasi (Projects, Skills, Contact), dan tombol "Download CV". |
| **Hero Section** | Foto profil profesional, Headline menarik, sub-headline (peran Anda), dan tombol CTA utama "Hire Me". |
| **About Me** | Cerita singkat tentang latar belakang, visi, dan apa yang Anda tawarkan sebagai profesional. |
| **Tech Stack / Skills** | Ikon teknologi yang Anda kuasai (React, Laravel, Figma, dsb). |
| **Portfolio / Projects** | Grid kartu proyek dengan gambar, deskripsi singkat, teknologi yang digunakan, dan link (Demo/GitHub). |
| **Experience & Edu** | Timeline riwayat pekerjaan dan pendidikan terakhir. |
| **Contact / Footer** | Form input nama & pesan yang ujungnya mengarah ke WhatsApp, serta link media sosial (LinkedIn, GitHub). |

## 3. App Summary untuk AI Coding

> **Project Name:** Professional Portfolio Landing Page with WhatsApp Integration
> **Tech Stack:** React (Next.js), Tailwind CSS, Framer Motion (for animations), Lucide React (for icons).
> **Core Objective:** Build a clean, high-converting, and responsive one-page portfolio for a [Sebutkan Peran Anda, misal: Fullstack Developer].
> **Key Features & Sections:**
> 1. **Hero:** Catchy headline, professional photo, and a primary CTA "Contact via WhatsApp".
> 2. **Portfolio Grid:** Display cards with image hover effects, project title, tech tags, and two buttons (View Demo & Source Code).
> 3. **Skill Icons:** A clean grid or a moving marquee of tech icons (e.g., React, Node.js, Tailwind).
> 4. **WhatsApp Integration Logic:** Create a floating WhatsApp button and a contact form. When the "Send" button is clicked, it should redirect to `https://wa.me/[NomorHPAnda]?text=[EncodedMessage]`.
> 5. **Sticky Navbar:** Smooth scroll navigation to sections.
> 6. **Dark/Light Mode:** Support for both themes.
> 
> 
> **Design Aesthetic:** Modern, minimalist, use professional colors (like Deep Blue or Slate), high-quality typography (Inter or Montserrat), and subtle entrance animations.
> **Mobile Optimization:** Ensure the layout is perfectly responsive for mobile viewing.

### Tips Tambahan untuk WhatsApp Integration:

Untuk membuat link WhatsApp yang rapi, gunakan format ini di kode Anda:
`https://wa.me/628123456789?text=Halo%20[Nama],%20saya%20tertarik%20dengan%20portofolio%20Anda.`

*Ganti `628123456789` dengan nomor Anda (mulai dengan kode negara tanpa tanda +).*
